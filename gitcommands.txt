Create a local folder, go into the folder and right click intialise git bash

Initialise git
    git init 

Set the origin 
    git remote add origin "repository link"                 // -u originname

show which files are added to index and ready to commit
    git status

Add a file to staging area 
    git add filename.ext

    add all files - git add . or git add -A 

Commit the changes 
    git commit -m "message"
